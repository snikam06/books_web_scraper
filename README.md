# books_web_scraper
Web scraping project using Python and BeautifulSoup

## Website
https://books.toscrape.com

## Data Extracted
- Book title
- Price
- Rating
- Availability
- Product link

## Tools Used
- Python
- BeautifulSoup
- Requests

## How to Run
- pip install requests beautifulsoup4
- python e-commerce_website.ipynb

## Purpose
Created for learning and portfolio building.
